# VideoMaker

> Automacao para criacao de videos com Node.js, integrando IBM Watson (NLP), Algorithmia (imagens) e YouTube API. Inspirado no canal Filipe Deschamps.

## Stack

- Node.js

## Prerequisitos

- Node.js v14+

## Como executar

```bash
# Clone o repositorio
git clone https://github.com/meirelesdev/VideoMaker.git
cd VideoMaker

# Instale as dependencias
npm install

# Inicie a aplicacao
npm start
```
